# 🌌 ATMOS_SECURE // Spatial Key Generator

An elegant, high-fidelity password generator inspired by the premium, dark aesthetics of **Dolby Atmos** mixed with modern **Glassmorphic** UI design principles. 

ATMOS_SECURE leverages cryptographic entropy (`window.crypto.getRandomValues`) to generate highly secure keys surrounded by a floating, multi-dimensional dark workspace.

---

## ✨ Features

- **Spatial Atmosphere UI:** Dark obsidian color palette accented with neon blues and hyper-purples mirroring high-end audio hardware interfaces.
- **Glassmorphic Architecture:** Beautiful frosted-glass card overlay with deep multi-layered shadows and real-time backdrop blur filtering (`backdrop-filter`).
- **Cryptographic Security:** Uses the browser's hardware-backed Web Crypto API for secure, truly random byte distribution.
- **Dynamic Feedback:** Features a real-time entropy strength meter that tracks password composition rules and length variables.
- **Smooth Clipboard integration:** Instant single-click copying with interactive visual confirmation states.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture.
- **CSS3:** Custom properties, CSS Grid layout, native range inputs, keyframe pulse animations, and backdrop blur.
- **JavaScript (ES6+):** Vanilla script featuring Web Crypto API integration.

---

## 🚀 Getting Started

Since this is a client-side vanilla web application, there are no dependencies to install. 

### Quick Launch
1. Clone or download this repository.
2. Ensure you have the three core files in the same directory:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` directly in any modern web browser (Chrome, Safari, Firefox, Edge).
